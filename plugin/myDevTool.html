# 插件的功能介绍

# 更改Cookie

更改Cookie的这个功能只能在AI助力科研 (opens new window)和http://loaclhost页面可用，在其他页面中按钮会处于禁用状态，如下图1所示。因此如果想要更改别的页面的Cookie是不可行的，这个功能开发出来本身就是为了写项目用的(几乎99%的情况下可用忽略该功能)，同时该功能只能获取Cookie字段中指定键值(JSESSIONID)所对应的值。 该功能只适合以下条件人群：

  1. 是AI助力科研的开发者之一
  2. 能够登陆AI助力科研页面

如果符合以上条件那么就可以看接下来该功能的使用介绍，否则就可以跳到下一部分。该功能的操作分为如下两步：

  1. 打开AI助力科研页面并登陆到主页中，点击获取Cookie按钮，那么输入框内就会出现一串字母并提示获取成功
  2. 切换到localhost页面，点击设置Cookie按钮，并提示设置成功

如果完成后就可以进入主页面了，挺简单的，相比于之前繁琐的步骤这样就显得非常方便快捷了。

点击观看示例视频

# 颜色吸管

使用方法：

  1. 点击获取颜色按钮，会出现一个放大镜
  2. 在网页中找到想取色的部分，单击鼠标
  3. 文本框会出现颜色的十六进制形式，也可以点击文本框最后的转换图标，这样就能在RGB形式与十六进制形式来回切换
  4. 点击复制rgb按钮，即可复制成功

# 设置默认路径 1.0.2+

这个功能与更改Cookie的条件一致，只适合符合条件的用户。 使用方法如下：

  1. 默认情况下的路径为index
  2. 点击右侧开关，开启功能
点击观看示例视频

TIP

想要使用此功能必须打开右侧的开关，打开后输入框禁止更改路径，想要更改就必须关闭功能，且该功能仅在localhost页面有效, 且生效后打开该页面会自动跳转至相应的默认路径下, 路径无需以/开头，默认情况下是index，也就是说功能开启后，当检测到当前页面为http://localhost/*时会自动跳转到http://localhost/index页面，如果将路径设置为其他值的话同理。

# 简易PS 1.0.3+

点击下方的跳转至图片裁剪页面按钮，会跳转至图片裁剪页面。总体页面如下方的图5所示。该页面包括了以下三个功能：

  1. 将图片转换成灰度图
  2. 将图片进行大小转换，在转换过程中可以对图片剩余部分填充的颜色以及透明度进行设置
  3. 保存图片
点击观看示例视频

TIP

可以点击生效区域或者将图片拖动至生效区域即可完成图片选择的功能。
