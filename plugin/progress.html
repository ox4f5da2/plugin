# 阅读进度条

该插件可以在每个页面的最顶端进行阅读进度展示，也就是会随着滚动条的位置不断改变进度条的长度。

# 大纲目录 1.1.0+

只在CSDN页面，将鼠标移动到页面的最右侧会弹出大纲目录，点击可进行快速跳转切换，点击大纲目录可以一键返回顶部。
