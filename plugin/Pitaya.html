# 全新的界面

安装后即可使用全新的百度界面新皮肤，但是如果想恢复到默认的百度界面，需要在Chrome浏览器的扩展程序页面关闭插件，具体如下图所示：

TIP

插件安装后该功能默认处于关闭状态，记得手动打开

# 去除百度广告

只要打开插件的使用开关即可开启去除百度广告，效果如图1所示

# 去除CSDN广告

只要打开插件的使用开关即可开启去除CSDN左侧的广告区域，并且将博客区域的宽度进行了拉伸，如图5所示

# CSDN链接跳转

以前点击CSDN的链接都会弹出一个确认界面，如下图所示，询问是否跳转，只要打开这个链接跳转功能，就能直接跳转到指定界面而不用每次再点击确定按钮了

点击观看示例视频

# CSDN关注博主查看全文破解 1.1.2+

只要打开是否打开屏蔽关注即可阅读功能后面的开关即可实现

下面提供一个测试页面 (opens new window)的入口链接，可以在该页面测试对应的功能

# 滚动翻页

开启功能后当滚动条滚动到底部或顶部时会自动翻页
