# 设置休眠模式样式

点击右上角的设置按钮，在下拉菜单中选择对应的模式(分为Default、当前时间、恋爱时间)即可设置休眠模式下的样式。样式的三种模式下的状态如下所示：

TIP

图1是Default模式，图2是当前时间模式，图3是恋爱时间模式

# 搜索引擎 1.0.5+

该功能可以在Baidu、Google、Bilibili、Github、谷歌学术进行搜索，点击搜索框右侧的文字进行搜索引擎的切换，并在输入框内输入想要搜索的文字，按回车即可进行搜索。

点击观看示例视频

# 搜索提示 1.0.6+

在输入框输入文字时会有文字的搜索提示功能，该提示使用的是百度搜索提示，有时候会无效，那么按照如下操作进行即可：

  1. 点击右上角的图标
  2. 点击点击测试按钮，在接口测试结果处会显示接口是否可用的状态
  3. 如果显示FAIL代表不可用，如果显示SUCCESS代表接口可用
  4. 如果在步骤3中显示不可用，只要一直点击按钮知道显示成功即可
点击观看示例视频
