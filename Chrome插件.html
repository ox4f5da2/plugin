# Chrome插件

这是关于开发Chrome插件的使用说明文档，目前完成了以下插件：

  • MyDevTool —— 拥有快速更改cookie、颜色吸管工具、简易PS功能
  • myNewTab —— 让你拥有一个焕然一新的newTab页面
  • Pitaya —— 拥有去除百度和CSDN页面中广告的功能，同时也能让你拥有一个全新的百度界面
  • 关于Chrome插件的开发，写起来挺复杂的，其实主要还是应该看看官方文档，然后自己百度实践一下，然后manifest文件的配置可以看右上角的相关内容

MyDevTool是帮助目前进行的项目快速开发而写的，myNewTab和Pitaya是给对象写的，然后就把最开始的需求丰富了一些，在她的需求下又加了点功能。以下是最开始的对话内容，从这以后就开始了长达一周的Chrome插件学习➕改BUG的过程：

  • 我：想要什么插件吗？
  • 她：我想把百度的广告去除了。
  • 我：好的呢！

同时还提供了manifest V3版本的更新文档、vuepress的简单使用文档、插件下载通道，如果更新其他的文档会放在右上角的Guide处

# 插件安装方式

# Chrome网上应用店安装

进入Chrome网上应用店 (opens new window)，在左上方的搜索框内输入插件名字搜索后安装即可

# Crx文件安装

  1. 下载对应版本的crx文件

TIP

在每种插件的介绍文档中Crx文件下载部分, 选择喜欢的版本后点击单元格内的按钮下载

  1. 打开Chrome浏览器，点击右上角的三个小点
  2. 选择更多工具
  3. 选择拓展程序
  4. 将crx文件拖入即可安装
