# http状态码

状态码 描述
200 服务器已成功处理了请求**(成功)**。
201 请求成功并且服务器创建了新的资源**(已创建)**。
204 服务器成功处理了请求，但没有返回任何内容**(无内容)**。
301 请求的网页已永久移动到新位置**(永久移动)**。
302 服务器目前从不同位置的网页响应请求，但请求者应继续使用原有位置来进行以后的请求**(临时移动)**。
304 自从上次请求后，请求的网页未修改过**(未修改)**。
400 服务器不理解请求的语法**(错误请求)**。
401 请求要求身份验证**(未授权)**。
403 服务器拒绝请求**(禁止)**。
404 服务器找不到请求的网页**(未找到)**。
409 服务器在完成请求时发生冲突**(冲突)**。
500 服务器遇到错误，无法完成请求**(服务器内部错误)**。
502 服务器作为网关或代理，从上游服务器收到无效响应**(错误网关)**。
503 服务器目前无法使用(由于超载或停机维护)(服务不可用)。
